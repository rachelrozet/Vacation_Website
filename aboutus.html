How the Website was Created

Bootstrap / Source Code with Modifications

  • Similar to other coding projects, Rachel utilized example source code from various websites, including Bootstrap and W3Schools, to begin building her application.
  • All source code was altered to support the needs of the final website.
  • For example, the [Subscribe] page of this website came from source code for entering a manufacturing order for a specific product. The source code was altered to rather support the entering of contact information for a vacation newsletter.
  • Other changes to source code include CSS/style changes to ensure code templates matched the website's theme.

Technologies Used

HTML, CSS, JavaScript

  • HTML
    • Provided the structure of each website page, including column classes, headers, paragraphs, lists, etc.
  • CSS
    • Used both in-line for each HTML page and in a separate style.css file, referenced in certain HTML pages that required similar style changes
    • Pages with specific and/or conflicting style parameters (section sizes, etc.) did not reference the style.css file
  • JavaScript
    • Most pages referenced a single script.js file. Functions in the file JavaScript include reading, inserting, updating, and deleting data from the [Subscribe] page, along with activating the navigation bar between pages.
    • In-line scripts and additional source scripts were required for the [Analytics] page to support individual chart.js visualizations
